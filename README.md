# MovieHub

MovieHub is a modern web application for discovering and exploring movies, powered by the TMDB API. Users can browse popular, top-rated, and upcoming movies, view detailed information about each film, including cast, production companies, and trailers.

---

## Technologies

- React + TypeScript  
- Vite  
- Tailwind CSS  
- TMDB API  
- Netlify (production deploy)  

---

## Demo

[Live Demo](https://darkos-movie-app.netlify.app/)  

---

## Key Features

- Browse popular, top-rated, and upcoming movies  
- Detailed movie pages with cast, production companies, and trailers  
- Movie search functionality  
- Responsive UI with skeleton loading for smooth experience
